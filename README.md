# 🎬 Movie Insights

Este proyecto contiene un dashboard interactivo desarrollado con **Streamlit** para el análisis exploratorio de datos de películas del dataset  (2020–2025).

---

## 📊 Descripción

Este proyecto permite visualizar y analizar datos de películas, explorando:

-  Tendencias de películas por año  
-  Popularidad y métricas de interacción  
-  Correlaciones entre variables  
-  Filtros interactivos  

El objetivo es aplicar técnicas de **análisis de datos (EDA)** y visualización para obtener insights relevantes.

---

## Tecnologías utilizadas

- Python 
- Streamlit 
- Pandas 
- Plotly
